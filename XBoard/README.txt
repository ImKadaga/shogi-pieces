Setup in XBoard:


XBoard Menu - > View -> Edit Themes List -> add these lines (of course change PATH_TO to the correct path where the extracted directories are):


"shogi kanji light (Ka)" -ubt true  -lbtf PATH_TO/board/wood1.png -dbtf PATH_TO/board/wood1.png -pid PATH_TO/kanji_light
"shogi kanji brown (Ka)" -ubt true  -lbtf PATH_TO/board/wood2.png -dbtf PATH_TO/board/wood2.png -pid PATH_TO/kanji_brown
"shogi kanji red wood (Ka)" -ubt true  -lbtf PATH_TO/board/wood4.png -dbtf PATH_TO/board/wood4.png -pid PATH_TO/kanji_red_wood
"shogi 2-kanji red wood (Ka)" -ubt true  -lbtf PATH_TO/board/wood4.png -dbtf PATH_TO/board/wood4.png -pid PATH_TO/2-kanji_red_wood
"shogi military (Ka)" -ubt true  -lbtf PATH_TO/board/military.png -dbtf PATH_TO/board/military.png -pid PATH_TO/military
"shogi international (Ka)" -ubt true  -lbtf PATH_TO/board/wood1.png -dbtf PATH_TO/board/wood1.png -pid PATH_TO/international
"shogi doubutsu (Ka)" -ubt true  -lbtf PATH_TO/board/doubutsu.png -dbtf PATH_TO/board/doubutsu.png -pid PATH_TO/doubutsu


Then you can change the board at View -> Board
